# 万象积木 使用指南

## 💬 对话

### 基本对话

在输入框输入消息，按 Enter 或点击「发送」开始对话。

### 快捷示例

点击输入框上方的快捷按钮快速填入示例问题：
- 💻 写代码
- 📊 数据分析
- 🔍 技术调研
- 📖 知识问答

### 多会话管理

- **新建会话**：点击「➕ 新会话」
- **切换会话**：从下拉列表选择已有会话
- **删除会话**：选中会话后点击「🗑️ 删除」

每个会话有独立的对话历史和记忆上下文。

### 重新生成

发送消息后，会自动显示「🔄 重新生成」和「✏️ 编辑上条」按钮。

### 设置

右侧控制面板：
- **选择模型**：切换使用的 LLM 模型
- **温度**：控制输出创意程度（0=精确，2=创意）
- **系统提示词**：定义 AI 的角色和行为

---

## 📚 知识库（RAG）

### 上传文档

1. 切换到「知识库」Tab
2. 选择文件（支持 PDF/Word/Markdown/HTML/TXT/CSV/Excel/PPT）
3. 点击「上传并索引」

### 知识库问答

在知识库 Tab 的查询框中输入问题，点击「查询」获取基于知识库的回答。

## 🧠 记忆系统

### 四层记忆架构

| 层级 | 名称 | 存储位置 | 生命周期 |
|------|------|----------|----------|
| L1 | 工作记忆 | 内存 | 会话期间 |
| L2 | 短期记忆 | JSON 文件 | 数小时/天 |
| L3 | 长期记忆 | ChromaDB | 数周/月 |
| L4 | 程序记忆 | JSON 文件 | 持久 |

### 自动记忆

系统会自动记住你说的话：

- **偏好**："我喜欢Python" → 存为偏好记忆
- **事实**："我叫张三" → 存为事实记忆  
- **事件**："我下周要去北京" → 存为事件记忆

下次对话中，AI 会自然地引用这些记忆。

### 记忆管理

在「🧠 记忆」Tab 中：
- 查看所有长期记忆
- 搜索特定记忆
- 删除记忆

---

## 📊 系统仪表盘

在「📊 系统」Tab 中查看：

- **系统概览**：各子系统运行状态
- **熔断器状态**：各模型的熔断/恢复状态
- **安全统计**：注入拦截/速率限制统计
- **审计事件**：最近的操作日志

## 🧩 Skill 插件

### 查看已安装 Skill

```bash
python -m cli skill list
```

### 创建 Skill

```bash
python -m cli skill create my-tool
# 编辑 my-tool/manifest.json 和 my-tool/handler.py
```

### 安装 Skill

```bash
python -m cli skill install ./my-tool
```

### 卸载 Skill

```bash
python -m cli skill uninstall my-tool
```

## 🤖 Agent 编排

系统会根据用户消息自动识别意图并分配合适的 Agent：

| 意图 | Agent | 说明 |
|------|-------|------|
| 写代码/调试 | CodeAgent | 代码生成、审查、调试 |
| 数据分析 | DataAgent | 数据分析、可视化、报表 |
| 技术调研 | ResearchAgent | 多源信息检索与整合 |
| 通用对话 | GeneralAgent | 日常对话、翻译、摘要 |
